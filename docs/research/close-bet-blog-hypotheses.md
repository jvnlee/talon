# 종가베팅 통설 가설 등록부 — 블로그 복기 글에서 추린 미고려 논점

2026-07-15 작성. 출처: 네이버 블로그 [종가배팅에 대해 깊게 공부하고 적용하자](https://blog.naver.com/buyingbymysalary/223497898669) (buyingbymysalary, 2024-07-01 게시). 유튜브 종가매매 영상들(창원개미·침착해·신정재·리노)을 개인이 복기한 노트라서, 근거 수준은 전부 **[미검증·통설]** — 수치·표본·검증이 하나도 없는 실전 경험담이다.

이 문서의 성격:

- **가설 등록부**다. 여기 적힌 것은 규칙이 아니라 검증 대상이며, 검증 전에는 어떤 항목도 전략 명세·라이브에 들어가지 않는다.
- talon이 **이미 고려한 논점은 뺐다** (`close-betting.md`·`close-bet-v1.md`와 겹치는 것). 원칙과 충돌하거나 물리적으로 불가능한 것은 경계 표에만 남겼다.
- **임계값 격자는 여기서 선언하지 않는다.** 격자 선언 = 사이클 개시 = N 카운트 시작이므로, 검증 착수를 승인하는 시점에 별도 선언한다 (v1 명세 7장 규율 그대로). → 2026-07-15 착수 승인에 따라 아래 "0단계 선언" 섹션에 확정했다.

## 용어

- **confluence (= interaction effect, 상호작용 효과)**: 신호 하나하나는 약한데 겹치면(합류하면) 효과가 커지는 현상.
- **ablation**: 결합에서 성분을 하나씩 빼 보며 각 성분이 실제로 기여하는지 재는 방법.
- **코호트(cohort)**: 어떤 조건을 만족해 뽑힌 (종목, 날짜)들의 집합. "그 조건이 고른 무리".
- **기간조정**: 크게 오른 뒤 며칠~몇 주 쉬면서 물린 물량(매물)이 소화되는 구간.
- **양매수**: 외국인과 기관이 같은 날 둘 다 순매수.
- **시간외단일가**: 16:00~18:00에 당일 종가 ±10% 안에서 10분 단위 단일가로 체결되는 장 마감 후 시장.
- **증거금 100% 종목**: 증권사가 외상(신용·미수)을 전혀 허용하지 않는 종목. 위험 종목이라는 간접 신호.

## 가설 목록

### H1. 기간조정 후 재돌파 코호트 — 선정 기준 자체를 바꾸는 가설

- **주장**: 급등 당일 추격하지 말고, 신고가(전고점) 돌파 뒤 10~40일 조정으로 매물을 소화하고 20일선 지지를 지킨 종목이 **거래량과 함께 전고점에 재접근하는 날** 종가 매수하라. 재료 없는 급등주는 고점에 물린다.
- **왜 관심**: v1 전멸의 원인이 "당일 상승폭 최대 3종목" 스코어가 +21% 급등 코호트만 골랐고 그 코호트는 비용 이전에도 엣지가 0이었다는 것. 이 가설은 정확히 **반대 코호트**(쉬었다가 재점화)를 고른다. v1 부검 결론과 방향이 맞물리는 유일한 외부 힌트.
- **코드화 스케치** (기존 15연산자로 표현 가능, 15:10 상태는 v1 근사 규칙 적용):
  - 최근 1년 최고가가 `cool_max`일 안에, 그러나 `cool_min`일 밖에 세워짐: `Max(high, cool_max) == Max(high, 250)` 이고 `Max(high, cool_min) < Max(high, 250)`
  - 조정 중 지지: `Min(close, cool_min) >= Mean(close, 20) * (1 - slack)`
  - 재접근: `close / Max(high, 250) >= near` + v1 조건 1·2(상승·거래량 배수) 유지
- **데이터**: KRX 일봉 이력으로 충분. **지금 검증 가능.**

### H2. 장대양봉 후 5일선 지지

- **주장**: 장대양봉 뒤 종가가 5일선 위에서 버티는 종목은 큰 시세로 이어지는 경우가 많다. 다음날 대량거래 음봉이 나와도 종가가 5일선을 지키면 후보 유지, "깔끔한 양봉으로 지지를 확인해 주는 날"이 진입일.
- **코드화 스케치**: 최근 `k`일 내 장대양봉 발생(등락 ≥ `big`, 몸통 비율 ≥ `body`) + 그 뒤 매일 종가 ≥ 5일선(`Min(close - Mean(close, 5), k) >= 0`) + 당일 15:10 상태 상승.
- **데이터**: 일봉으로 충분. **지금 검증 가능.**

### H3. 시간외단일가 등락 → 익일 갭 예측력

- **주장(원문 연쇄)**: 장후 시간외(15:30~16:00) 매수세 강함 → 시간외단일가(16~18시) 상승 → 익일 시가 갭업 확률↑.
- **검증 범위**: 앞 고리(15:40~16:00 매수/매도 잔량)는 실시간 호가라 수집·백필 모두 불가 — **검증 불가로 못 박는다.** 가운데→끝 고리(시간외단일가 등락 → 익일 갭)는 **overtime 잡(18:10)이 2026-07-15부터 종목별 시간외 OHLCV·등락률(당일 종가 대비)을 저장하므로 그대로 검증 가능.** talon이 이 가설을 위해 유일하게 기록 중인 데이터다.
- **대상 모집단**: `auction_symbols` = 15:10 결정 스냅샷 후보(없으면 pinned). 우리가 실제 결정에 쓰는 모집단이라 오히려 적합.
- **방법**: 익일 갭(익일 시가 ÷ 당일 종가 − 1)의 조건부 분포를 시간외 등락률 구간별로 비교. 예측력이 있으면 → (a) 사이징의 갭 가정 정교화, (b) v2 보유/청산 게이트 후보, (c) 17~18시 알림(퇴근 후 유일한 실시간 대응 창구) 후보.
- **시점**: 과거치 백필 불가 → 전방 적립만. 하루 후보 3~10종목 × 60~120거래일이면 1차 판독 (**약 3~6개월 후**). 그때까지 할 일은 "계속 쌓기"뿐이고 이미 돌아가고 있다.

### H4. 외국인·기관 양매수 필터

- **주장**: 대형주 종가베팅은 외인·기관 양매수를 반드시 확인하라.
- **데이터 제약**: 15:10 잠정수급은 라이브 참고용일 뿐 **백테스트 투입 금지**(잠정치 원칙). 검증에는 KRX 확정 수급 이력이 필요 → **B그룹 백필 결정 대기.** 이 가설은 백필 우선순위에서 수급을 올릴 근거가 된다.
- **코드화 주의**: 당일 확정 수급은 15:10에 존재하지 않는다. 라이브·백테스트 동형성을 위해 신호는 **전일(D−1)까지의 확정 수급**으로 정의해야 정직하다 (글은 당일치를 보지만 우리는 볼 수 없다).

### H5. 시총이 낮을수록 갭이 크다 (조절 변수 가설)

- **주장**: 같은 재료라면 시총 낮은 종목이 갭 뜰 확률이 높다.
- **코드화 스케치**: 동일 신호 코호트를 시총 3분위로 갈라 익일 갭 분포 비교. 신호 × 시총의 상호작용 검증이라 confluence 프로토콜(아래)의 표준 예제로 쓰기 좋다.
- **데이터**: `marketcap` 스토어가 일일 수집·재조정 중. **IS 구간(2016~2023) 시총 이력 백필 여부만 확인 필요** (pykrx로 백필 용이).

### H6. 증거금 100% 종목 제외 (유니버스 필터)

- **주장**: 창원개미 — 증거금 100% 종목은 종가베팅에서 제외한다.
- **문제**: 증거금률은 증권사별 지정이라 과거 이력 백필이 사실상 불가. 공식 대체재는 **KRX 시장경보(투자주의/경고/위험) 이력** — 경고·위험 지정에는 위탁증거금 100% 징수가 따라오므로[미검증·확인 필요] 상당 부분 겹친다. v1 유니버스는 관리·환기만 배제하므로 경보 단계 배제는 신규다.
- **상태**: KRX 시장경보 이력의 백필 경로(정보데이터시스템/Open API) 확인 필요.

### H7. 엔벨로프(20,40) 돌파 후 7·15일선 지지 — 보류

- 원문 "엔벨로프 20,40선"의 파라미터 정의(기간 20에 폭 40%인지, 선 두 개인지)가 글만으로 불명. 원 영상을 확인하기 전에는 코드화 불가 → **보류.** 7·15일선 지지 부분은 H2의 변형으로 흡수 가능.

## 검증 프로토콜 초안 — confluence를 정식 절차로

v1의 교훈: 조건 3개를 AND로 묶고 격자 27개를 태웠지만, **각 성분의 단독 엣지도, 결합이 단독 대비 얼마나 보태는지도 잰 적이 없다.** "결합이 고른 코호트 자체에 엣지가 없다"는 사실을 백테스트 27번 뒤에야 알았다. 이번에는 순서를 뒤집는다.

**0단계 — 단일 신호 진단 (IS만, 비용 전, 패널 집계)**
각 가설을 신호 하나로 코드화하고, 신호 발화 코호트의 익일 갭(종가→시가) 분포를 무신호 기준선과 비교한다. 백테스트 엔진이 아니라 패널 집계로 — 빠르고, 비용·체결 가정이 안 끼어 "원재료에 엣지가 있는가"만 본다. 결과(신호일 수, 평균·중앙값 갭, 기준선 대비 차이)는 전부 trials에 기록한다 — **진단도 선택편향을 만들므로 N에 센다.** 방향이 0이거나 음이면 그 가설은 여기서 탈락. v1이었다면 이 단계에서 걸렸다.

**1단계 — 결합 증분 측정 (confluence)**
0단계 생존 신호만 2~3개씩 AND 결합. **증분 = 결합 코호트 엣지 − max(성분 단독 엣지).** 증분이 표본 크기 대비 유의해야 "합류 효과 있음"이다. ablation으로 성분을 하나씩 빼며 엣지 하락폭을 확인하고, 기여 없는 성분은 노브만 늘리므로 제거한다.
경고: **조합은 N을 지수적으로 태운다.** 신호 k개면 단독 k + 쌍 k(k−1)/2 + 삼중 조합까지 금방 수십 개다. k ≤ 5로 못 박고, 시험한 모든 집계를 기록한다.

**2단계 — 전략 승격**
1단계 최종 결합 1~2개만 v2 명세로 작성 → 승인 → 격자 선언(N 확정) → 비용 포함 엔진 백테스트 → DSR → 관문 1. (v1과 동일한 길, 시작점만 다르다.)

## 0단계 선언 — 사이클 `blog-stage0` (2026-07-15 착수 승인)

이번 사이클에서 시험할 정의를 여기 못 박는다. **아래에 없는 변형·임계값은 이번 사이클에서 시험하지 않으며**, 아래 집계 각각을 trial로 기록한다. 총 11건: H1, H2, 그리고 시총 3분위 분할(H1×3, H2×3, 기준선×3).

### 공통 프로토콜

- 구간: 2016-07-01 ~ 2023-12-31, OOS 클램프(`clamp_is_end`, 봉인 경계 2024-01-01) 강제. 250+40일 워밍업으로 신호 유효 시작은 대략 2017-09.
- 유니버스: v1과 동일 — `tradable_stock` + 최근 20일 평균 거래대금 상위 300.
- 결과 변수(익일갭): 종목별 다음 거래봉 시가 ÷ 당일 종가 − 1. 다음 봉이 달력일 7일 넘게 떨어져 있으면(거래정지) 제외하고 건수 보고.
- 코호트 제외: 신호일 종가가 상한가 근사(전일 종가 대비 +29% 이상)면 제외(동시호가 매수 불가) — 건수 보고.
- 보고 통계(코호트·기준선 동일): n, 평균, 중앙값, 표준편차, 승률(갭>0), p10, p90.
- 기준선: 유니버스 전체 (종목, 날짜) 쌍.
- **생존 기준(사전 선언)**: Δ평균(코호트 − 기준선) > 0 이고 Welch t ≥ 2 이고 n ≥ 100. n < 100이면 판정 보류.
- 신호의 15:10 상태는 v1 명세 7장의 일봉 근사 규칙을 상속한다 (가격 잡음 ±0.5%, 거래량 f-비중 주의).

### H1 정의 — 기간조정 후 재돌파 (조각 전부 AND)

`PH`는 표기 편의이고 구현은 인라인 전개한다.

| 조각 | 표현식 |
|---|---|
| 전고(1년, 당일 제외) | `PH = Ref(Max(high, 250), 1)` |
| 냉각 10~40일 | `Ref(Max(high, 40), 1) >= PH` 그리고 `Ref(Max(high, 10), 1) < PH` |
| 조정 중 지지 | `Min(close - Mean(close, 20) * 0.97, 10) >= 0` |
| 전고 재접근 | `close >= PH * 0.95` |
| 당일 상승 | `close > prev_close` |
| 거래량 재점화 | `volume >= Ref(Mean(volume, 20), 1) * 1.5` |

### H2 정의 — 장대양봉 후 5일선 지지 (조각 전부 AND)

| 조각 | 표현식 |
|---|---|
| 장대양봉(직전 10일 내, 당일 제외) | `BC = If(close >= Ref(close, 1) * 1.10, 1, 0) * If(close - open >= (high - low) * 0.6, 1, 0)`, 조건은 `Ref(Max(BC, 10), 1) >= 1` |
| 5일선 지지 | `Min(close - Mean(close, 5), 5) >= 0` |
| 당일 양봉·상승 | `close > open` 그리고 `close > prev_close` |

참고: 점상한가(고가=저가=종가)도 BC를 만족한다 — 의도된 포함.

### H5 분할

H1 코호트, H2 코호트, 기준선 각각을 **그날 유니버스 내 시총 3분위**(`marketcap` 스토어 당일값)로 갈라 동일 통계를 비교한다. 시총 이력은 2016-07-01부터 완비 확인(2026-07-15).

### 기록

`talon trials open blog-stage0`로 사이클을 열고 집계 11건을 trial로 기록한다. 직전 활성 사이클은 그대로 장부에 남는다(아카이브 취급).

### 판독

1. **H1(기간조정 후 재돌파)은 생존.** 다만 전체 t=+2.17은 11회 관측의 다중성을 감안하면 약하다. 강건한 발견은 아래 3번.
2. **H2(장대양봉 후 5일선 지지)는 문언 그대로는 기각.** 코호트가 넓고(하루 평균 ~7건 발화) 승률이 기준선보다 낮다. 단, 시총과 결합하면 중·대형에서 되살아난다 — **단일로는 죽은 신호가 결합에서 증폭되는 confluence의 실물 사례.**
3. **H5 통설은 신호 조건부에서 반전.** 무조건부 밤샘 갭은 소형이 최대(+0.402%)로 블로그 문언과 일치하지만, **신호의 증분 엣지는 정반대로 대형주에 집중**(H1×대형 t=+4.03, H2×대형 t=+4.36 — 다중성 보정에도 견디는 유일한 수치들). 소형주에서는 두 신호 모두 죽는다. 해석: 소형주는 신호 없이도 밤에 떠서 신호가 정보를 못 보태고, 대형주에서는 이 패턴들이 실제로 정보를 보탠다.
4. **절대 수익성 경고**: 생존 코호트의 평균 갭 +0.36~0.47%는 왕복 비용(0.25~0.4% 수준)과 같은 자릿수다. 0단계는 스크린이지 손익 증명이 아니다 — 비용 포함 판정은 2단계 엔진 백테스트에서만.

### 구현 충실도 노트 (적대 검증 3방향: blocker 0, minor 5)

- **유니버스는 v1 코드와 동일한 "당일 거래대금 상위 300" 랭크.** 이번 검증에서 v1 명세 문서(§1 "최근 20일 평균 거래대금 상위 300")와 v1 실제 코드가 서로 다름이 발견됐다 — v1 문서 부채, 별도 정리 필요.
- 상한가 제외는 선언 문언대로 코호트에만 적용(기준선 유지 — 기준선 평균을 끌어올리는 보수적 방향).
- 시총 분위는 결과값 있는 행 기준 랭크(정지가 희소해 영향 무시 가능). 등개수 분할, 동률은 하위 분위.
- `diagnose_cohorts`를 CLI 밖에서 단독 재사용할 때 end 경계가 자체 봉인되지 않는 잠재 함정 있음(현 CLI 경로는 `clamp_is_end`+사전 절단으로 봉인 확인됨).
- trial 11건은 N에 세지만 sharpe 분포에는 안 들어가 DSR을 오염하지 않는 방식으로 기록 (`blog-stage0` 사이클, id 68~78).
