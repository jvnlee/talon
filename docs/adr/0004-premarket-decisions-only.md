# 진입 결정은 장전에만 내리고, 장중은 방어 전용으로 운영한다

사용자는 직장인(09:30~17:00 근무)이라 한국 장중 상시 대응이 불가능하다. 장중 신규 진입 시그널은 (1) 실행률이 낮아 페이퍼 트레이딩 성과와 실제 성과를 괴리시키고, (2) 분봉 기반이라 한국 분봉 과거 데이터 부재로 백테스팅이 불가능하며, (3) 알림 노이즈로 방어 알림의 주의력을 잠식한다. 따라서 모든 진입 결정은 장전 파이프라인에서 확정해 장전 브리핑(08:20)으로 전달하고, 매매 결론에 손절가·목표가를 포함해 사용자가 주문 시점에 토스 조건주문으로 방어를 심을 수 있게 한다. 장중 에이전트는 감시 전용이며 보유 종목의 리스크 이벤트(손절·목표가 근접, 급등락, 긴급 공시)에 대한 방어 알림만 발송한다. 백테스트 진입 가정은 "다음날 시가"로 단순화한다.

## Consequences

- 하루 사이클: 07:30 장전 파이프라인 → 08:20 장전 브리핑 → 09:00~09:30 사용자 주문 → 장중 방어 감시 → 15:40 장마감 리뷰(체결 대조) → 저녁 리뷰·질의응답.
- 매매 결론과 실제 체결의 자동 대조(실행 대조)를 유지해 성과를 시스템 기준/실행 기준 두 트랙으로 기록한다.
- 자동매매 단계에서는 사람 실행 제약이 사라지므로, 분봉 데이터 축적·검증을 전제로 장중 진입을 재검토할 수 있다.
