# 매매 결론은 결정론적 퀀트 코어가 생성한다

talon의 검증 사다리는 백테스팅 → 실시장 페이퍼 트레이딩 → 실전 → 자동매매다. LLM의 재량 판단은 비결정적이고, 학습 데이터 누수(모델이 과거 시점 이후의 결과를 이미 알고 있음) 때문에 백테스팅이 성립하지 않으며, 구독제 사용량 한도로는 과거 재생 비용도 감당할 수 없다. 따라서 매매 결론(BUY/HOLD/SELL + 투자 비중)은 결정론적 코드인 퀀트 코어가 생성하고, LLM은 리서치·근거 해설·전략 제안 층으로 분리한다. 자동매매 단계로 가더라도 주문 실행 주체는 검증된 퀀트 코어다.

## Considered Options

- LLM 멀티에이전트가 최종 판단 (TradingAgents 방식) — 백테스팅 불가, 구독제 호출량 한도와 충돌, 자동매매 시 비결정적 판단이 주문에 직결되어 기각
- 듀얼 트랙 병행 운용 — 퀀트 코어 안정화 이후 LLM 그림자 트랙으로 점진 도입 가능한 상위 집합이므로 초기 선택지에서 제외
