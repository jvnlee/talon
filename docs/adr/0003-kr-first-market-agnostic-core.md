# MVP는 한국 시장만 검증하고, 코어는 시장 불가지론적으로 설계한다

사용자는 한국장 위주의 단타·스윙을 원하고, 토스증권이 주 브로커이며, 시드 1000만원을 두 시장에 나누면 포지션이 얇아진다. 따라서 백테스팅과 페이퍼 트레이딩 검증은 한국 시장만으로 시작한다. 다만 코어 파이프라인(데이터 → 팩터 → 시그널 → 비중)은 시장을 모르는 구조로 짜고 거래 캘린더·데이터 소스·통화·시장 미시구조(호가단위, 상하한가, 거래세)는 시장별 어댑터로 분리해, 미국 확장이 어댑터 추가로 끝나게 한다.

## Consequences

- 한국 분봉 과거 데이터는 무료 확보 경로가 없으므로, 분봉 실시간 누적 파이프라인을 프로젝트 첫날부터 가동한다.
- 미국 확장 시 백테스팅 데이터는 Alpaca 무료 티어(10년치 1분봉)를 사용할 수 있어 데이터 관점의 추가 비용은 낮다.
- 유니버스는 정적 종목 리스트가 아니라 동적 필터(거래대금·시가총액 하한, 관리종목·거래정지 제외)로 매일 재구성해 생존 편향을 차단한다.
