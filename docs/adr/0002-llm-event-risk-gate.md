# LLM은 이벤트 리스크 게이트로만 매매 결론에 관여한다

뉴스·공시 같은 정성 정보를 매매 결론에 반영하되 백테스팅 검증 체계를 깨지 않기 위해, LLM 레이어의 개입을 이벤트 리스크 게이트 하나로 제한한다. 퀀트 코어가 매매 결론을 생성한 뒤 LLM이 해당 종목의 이벤트 리스크(실적 발표 임박, 공시, 거래정지 위험, 급락 뉴스)를 점검하고, 비중 축소·진입 보류만 지시할 수 있다. 신규 매수 지시와 비중 확대는 불가하다. 게이트의 모든 개입은 로그로 남겨 페이퍼 트레이딩에서 게이트 유/무 성과를 비교해 개입 가치를 검증한다. 백테스팅은 퀀트 코어 단독으로 수행하며 게이트는 백테스트 대상이 아니다.

## Consequences

- 페이퍼 트레이딩 기간에 LLM의 뉴스 감성·이벤트 점수를 포인트인타임으로 축적하고, 충분한 포워드 데이터가 쌓이면 검증을 거쳐 퀀트 코어의 입력 팩터로 승격하는 진화 경로(정량 팩터 입력)를 열어둔다.
