# 검증 사다리의 관문 기준은 결과를 보기 전에 고정한다

단계 승격 기준을 사전에 수치로 고정해, 결과를 본 뒤 기준을 결과에 맞추는 판단 오염을 차단한다.

관문 1 — 백테스트 → 페이퍼 (전략별 심사):
- 최소 10년 일봉(2015 가격제한폭 ±30% 이후, 2018·2020·2022 하락장 포함), 최근 2~3년은 OOS로 봉인하고 단 한 번만 평가(OOS 성적을 보고 재튜닝 금지)
- 비용 전액 반영 후: OOS 수익률 > 동기간 KOSPI 매수후보유, MDD ≤ 20%, 트레이드 ≥ 200건, profit factor ≥ 1.3
- 과최적화 방어: 룩어헤드 자동검출 클린, 파라미터 민감도(인접 파라미터에서도 성립), Deflated Sharpe > 0

관문 2 — 페이퍼 → 실전:
- 최소 3개월 AND 최소 30 트레이드(미달 시 연장)
- 페이퍼 성과가 백테스트 OOS 기대 범위 내(현저히 좋아도 체결 모델 결함 의심 — 파리티 검증이 목적), 시스템 가동률 ≥ 98%, 실행 대조·손실 브레이커·방어 알림 실동작 확인, LLM 게이트 유/무 비교에서 게이트가 무해 이상
- 승격 후에도 단계 투입: 300만원 → 600만원 → 1000만원 (각 단계 1~2개월 정상 운영 확인)

관문 3 — 실전 → 자동매매: 실전 운영 데이터가 쌓인 시점에 별도 설계 세션으로 정한다(사전 확정하지 않음).
