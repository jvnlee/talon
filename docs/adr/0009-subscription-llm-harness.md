# LLM은 Claude Max 20x 구독의 공식 CLI 경로로만 호출한다

LLM 비용은 사용자의 Claude Max 20x 구독으로만 충당하며 종량 API는 절대 사용하지 않는다. 구현 경로: launchd가 정해진 시각에 headless Claude Code(공식 CLI 로그인 세션/Agent SDK)를 실행하고, 상태는 SQLite에 영속화한다("깨어남 → 상태 로드 → 작업 → 저장 → 종료" — 상주 프로세스 아님). 텔레그램 봇은 LLM 클라이언트가 아니라 공식 CLI 서브프로세스의 입출력 중계기다.

금지 사항: `ANTHROPIC_API_KEY` 설정(종량 과금 전락), OAuth 토큰 추출 후 서드파티 클라이언트 재사용(계정 정지 사례).

사용량 규율: Max 20x의 병목은 Opus이므로 다수 작업(뉴스 요약·데이터 정리)은 Sonnet으로, 핵심 판정(이벤트 리스크 게이트, 장전 브리핑 종합)만 Opus로 라우팅하고, 결정당 LLM 호출을 한 자릿수로 압축한다. 자체 사용량 로깅으로 비공개 한도를 캘리브레이션한다.

정책 헤지: Anthropic 자동화 과금 정책이 유동적이므로 LLM 호출부를 얇은 인터페이스 뒤에 두어 Codex CLI(ChatGPT 구독)로 스왑 가능하게 유지한다. Codex 구독은 현재 불필요하며 폴백 슬롯으로만 설계에 반영한다.
