# Talon

한국·미국 주식의 단기(일~주 보유)·스윙 매매를 위한 개인용 투자 에이전트. 시황·뉴스 리서치와 퀀트 시그널을 결합해 종목별 매매 결론을 제시하고, 사용자가 최종 매매를 실행한다.

## Language

**퀀트 코어 (Quant Core)**:
매매 결론을 생성하는 결정론적 코드 파이프라인. 같은 입력 데이터에 항상 같은 출력을 내며, 백테스팅 검증의 대상이 되는 유일한 판단 주체.
_Avoid_: 전략 엔진, 시그널 봇

**LLM 레이어 (LLM Layer)**:
뉴스·시황 리서치, 매매 결론의 근거 해설, 신규 전략 제안을 담당하는 층. 구독제(Claude Max) 기반으로 실행되며, 퀀트 코어의 결정론성을 해치지 않는 범위에서만 판단에 관여한다.
_Avoid_: AI 판단부

**매매 결론 (Recommendation)**:
퀀트 코어 파이프라인의 최종 산출물. 종목별 액션(BUY / HOLD / SELL)과 투자 비중으로 구성된다.
_Avoid_: 시그널(중간 산출물과 혼동), 주문(실행과 혼동)

**이벤트 리스크 게이트 (Event Risk Gate)**:
퀀트 코어의 매매 결론을 LLM이 이벤트 리스크 관점에서 점검하는 관문. 비중 축소와 진입 보류만 지시할 수 있으며, 모든 개입은 로그로 기록된다.
_Avoid_: LLM 필터, 뉴스 필터

**페이퍼 엔진 (Paper Engine)**:
백테스트 체결 시뮬레이터를 실시간 시세에 연결해 가상 잔고로 매매를 재현하는 검증 장치. 체결 가정은 항상 실전보다 보수적으로 설정한다.
_Avoid_: 모의투자(증권사 제공 서비스와 혼동)

**R 사이징 (R-based Sizing)**:
트레이드당 계좌 리스크 상한을 정하고 진입가-손절가 거리로 수량을 역산하는 포지션 사이징 방식. 모든 트레이드의 최대 손실이 균일해진다.
_Avoid_: 정액 배분, 균등 비중

**손실 브레이커 (Loss Breaker)**:
계좌의 실현 손실·낙폭이 한도에 도달하면 신규 진입을 중단하는 사후적 방어 장치. 보유 종목의 방어·청산은 계속 동작한다. KRX의 시장 전체 서킷브레이커 제도와는 무관하다.
_Avoid_: 서킷브레이커(KRX 제도와 혼동)

**국면 필터 (Regime Filter)**:
지수 추세·변동성으로 시장 국면을 판정해 전략별 가중과 전체 노출을 조절하는 상위 규칙. 모멘텀과 평균회귀처럼 국면에 따라 상성이 반대인 전략들의 스위치 역할을 한다.
_Avoid_: 마켓 타이밍, 시장 필터

**종가 베팅 (Close Bet)**:
장 마감 무렵(15:10경) 결정해 마감 동시호가에 매수하고 기본적으로 익일 시가에 청산하는 오버나이트 실행 모드. 밤사이 드리프트를 수취하는 것이 목적이며, 진입 결정이 장전이 아닌 오후 슬롯에서 내려지는 유일한 예외다(ADR 0012). **진입 신호는 전일까지의 완결 데이터로만 만든다** — 15:10에 당일 종가·고가·전체 거래량은 아직 존재하지 않는다(ADR 0013).
_Avoid_: 상따(상한가 따라잡기와 혼동), 데이트레이딩

**장전 브리핑 (Pre-market Briefing)**:
장전 파이프라인의 산출물로 개장 전 텔레그램으로 발송되는 보고. 종목별 매매 결론(진입가 범위·손절가·목표가 포함)과 시황·근거 해설로 구성된다.
_Avoid_: 모닝 리포트, 데일리 시그널

**방어 알림 (Defense Alert)**:
장중 감시가 보유 종목의 리스크 이벤트(손절·목표가 근접, 급등락, 긴급 공시)를 감지했을 때 발송하는 텔레그램 알림. 장중에 발송되는 유일한 알림 유형이다.

**실행 대조 (Execution Reconciliation)**:
매매 결론과 실제 체결 내역(토스 API 조회)을 자동 대조하는 절차. 성과를 시스템 기준과 실행 기준 두 트랙으로 분리 기록하는 근거가 된다.

**유니버스 (Universe)**:
매일 동적 필터로 재구성되는 매매 후보 종목 집합. 정적 종목 리스트가 아니다. 종목 분류는 KRX 공식 종목기본정보(`stock_info` 데이터셋)의 일자별 스냅샷을 정본으로 삼아 보통주만 남기고 리츠·외국주권·주식예탁증권·우선주·관리종목·SPAC·투자주의환기종목을 제외하며, 여기에 거래대금 하한과 상위 N 컷을 얹는다. 판정 규칙은 `quant/universe.py:tradable_stock()` 하나이고 라이브와 백테스트가 이를 공유한다.
_Avoid_: 종목 풀, 워치리스트

**종목기본정보 (Stock Info)**:
KRX Open API가 주는 일자별 종목 분류 스냅샷(`SECUGRP_NM`·`KIND_STKCERT_TP_NM`·`SECT_TP_NM`). 시점별 정본이라 과거 유니버스를 룩어헤드 없이 재구성할 수 있다. T+1로 확정되므로 reconcile 잡이 익일에 채운다.
_Avoid_: 종목마스터, 티커 메타

**단타**:
보유기간 1일~수 주의 단기 매매. 장중 고빈도 매매·스캘핑은 포함하지 않는다.

**스윙**:
보유기간 수 주 이상의 중기 매매.
